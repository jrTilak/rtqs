# Realtime Quiz System - Web Client

The frontend application built with **React** and **Vite**, providing an interactive interface for admins and players.

## Technology Stack

- **Core**: [React](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Language**: TypeScript
- **Styling**: [TailwindCSS](https://tailwindcss.com/)
- **UI Components**: [shadcn/ui](https://ui.shadcn.com/) (based on Radix UI)
- **State/Data Fetching**: [TanStack Query](https://tanstack.com/query) (React Query)
- **Routing**: [TanStack Router](https://tanstack.com/router)
- **API Client**: Orval (Auto-generated from Server OpenAPI)

## Features

- **Lobby Management**: Real-time status updates (Waiting, In Quiz, Summary).
- **Admin Controls**: Start questions, evaluate answers, view live responses.
- **Player Interface**: Answer submission, live timer/stopwatch, result summary.
